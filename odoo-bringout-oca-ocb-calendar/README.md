# Calendar


This is a full-featured calendar system.
========================================

It supports:
------------
    - Calendar of events
    - Recurring events

If you need to manage your meetings, you should install the CRM module.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-calendar
```

## Dependencies

This addon depends on:
- base
- mail
